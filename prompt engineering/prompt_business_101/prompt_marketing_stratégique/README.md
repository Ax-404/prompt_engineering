# Prompt Marketing - Bibliothèque de Prompts OpenClaw

Ce dossier contient 21 prompts marketing structurés pour OpenClaw, basés sur le document `marketing+clawbot.pdf` et les meilleures pratiques de prompt engineering.

## Structure

Chaque fichier correspond à un prompt marketing spécifique, structuré selon les principes du guide de prompt engineering :

- **Contexte** : Définition du rôle et de l'expertise
- **Objectif** : But clair et spécifique
- **Instructions Détaillées** : Étapes précises par phase
- **Skills Requis** : Compétences techniques nécessaires
- **Format de Sortie** : Structure des rapports et alertes
- **Contraintes** : Limitations et règles importantes
- **Résultat Attendu** : Bénéfices et outcomes

## Utilisation

Chaque prompt peut être utilisé directement avec OpenClaw. Il suffit de :

1. Ouvrir le fichier du prompt souhaité
2. Copier le contenu complet
3. L'envoyer à OpenClaw comme instruction système ou prompt de tâche
4. Adapter les paramètres spécifiques (noms de marques, URLs, etc.)

## Personnalisation

Avant d'utiliser un prompt, pensez à personnaliser :
- Les noms de marques/entreprises
- Les URLs et liens spécifiques
- Les critères de filtrage (niches, audiences, etc.)
- Les outils et intégrations (APIs, CRMs, etc.)
