# Prompt News - Bibliothèque de Prompts pour Veille Informationnelle

Ce dossier contient des prompts structurés pour la veille et l'analyse de différents types d'actualités, basés sur le guide de prompt engineering.

## Structure

Le dossier est organisé en 4 sous-dossiers correspondant aux différents domaines d'actualité :

- **economie_geopolitique/** : Économie mondiale, US, européenne et géopolitique
- **marches_financiers/** : Cryptomonnaies et marchés boursiers
- **finance_entreprise/** : Finance d'entreprise et analyse de résultats
- **ia_technologies/** : IA et nouvelles technologies

## Calendrier de Livraison

**⚠️ IMPORTANT : Chaque sous-dossier a un calendrier de livraison spécifique.**

| Dossier | Jour(s) de livraison | Fréquence |
|---------|---------------------|-----------|
| **economie_geopolitique** | **Lundi** | Hebdomadaire |
| **finance_entreprise** | **Mardi** | Hebdomadaire |
| **ia_technologies** | **Jeudi et Samedi** | Bi-hebdomadaire |
| **marches_financiers** | **Dimanche** | Hebdomadaire |

**Exceptions** : Les alertes critiques peuvent être envoyées immédiatement, indépendamment du calendrier prévu.

## Structure de Chaque Prompt

Chaque prompt suit la structure du guide de prompt engineering :

- **Contexte** : Définition du rôle et de l'expertise
- **Objectif** : But clair et spécifique
- **Instructions Détaillées** : Étapes précises par phase
- **Skills Requis** : Compétences techniques nécessaires
- **Format de Sortie** : Structure des outputs attendus
- **Contraintes** : Limitations et règles importantes
- **Résultat Attendu** : Bénéfices et outcomes

## Utilisation

Chaque prompt peut être utilisé directement avec OpenClaw ou tout autre agent AI. Il suffit de :

1. Ouvrir le fichier du prompt souhaité
2. Copier le contenu complet
3. L'envoyer à l'agent comme instruction système ou prompt de tâche
4. Adapter les paramètres spécifiques (sources, entreprises, secteurs, etc.)

## Personnalisation

Avant d'utiliser un prompt, pensez à personnaliser :
- Les sources spécifiques à surveiller
- Les entreprises/secteurs d'intérêt
- La fréquence de veille souhaitée
- Les critères d'alerte et de priorisation
