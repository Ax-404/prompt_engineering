# Prompt Social Media - Bibliothèque de Prompts pour Réseaux Sociaux

Ce dossier contient 10 prompts structurés pour booster le contenu sur les réseaux sociaux, basés sur le guide de prompt engineering et organisés par plateforme.

## Structure

Le dossier est organisé en 4 sous-dossiers correspondant aux principales plateformes :

- **reddit/** : 2 prompts pour l'engagement Reddit
- **X/** : 3 prompts pour X (Twitter)
- **newsletter/** : 2 prompts pour les newsletters
- **linkedin/** : 3 prompts pour LinkedIn

## Liste des Prompts

### Reddit (2 prompts)
1. **01_reddit_post_optimizer.md** - Création de posts Reddit optimisés et authentiques
2. **02_reddit_community_engagement.md** - Stratégie d'engagement communautaire Reddit

### X / Twitter (3 prompts)
1. **01_x_thread_creator.md** - Création de threads X engageants et viraux
2. **02_x_engagement_optimizer.md** - Optimisation d'engagement pour tweets
3. **03_x_content_calendar.md** - Calendrier de contenu X complet

### Newsletter (2 prompts)
1. **01_newsletter_writer.md** - Rédaction de newsletters engageantes
2. **02_newsletter_segmentation.md** - Stratégie de segmentation et personnalisation

### LinkedIn (3 prompts)
1. **01_linkedin_article_writer.md** - Rédaction d'articles LinkedIn professionnels
2. **02_linkedin_post_optimizer.md** - Optimisation de posts LinkedIn
3. **03_linkedin_engagement_strategy.md** - Stratégie d'engagement LinkedIn complète

## Structure de Chaque Prompt

Chaque prompt suit la structure du guide de prompt engineering :

- **Contexte** : Définition du rôle et de l'expertise
- **Objectif** : But clair et spécifique
- **Instructions Détaillées** : Étapes précises par phase
- **Skills Requis** : Compétences techniques nécessaires
- **Format de Sortie** : Structure des outputs attendus
- **Contraintes** : Limitations et règles importantes
- **Résultat Attendu** : Bénéfices et outcomes

## Utilisation

Chaque prompt peut être utilisé directement avec OpenClaw ou tout autre agent AI. Il suffit de :

1. Ouvrir le fichier du prompt souhaité
2. Copier le contenu complet
3. L'envoyer à l'agent comme instruction système ou prompt de tâche
4. Adapter les paramètres spécifiques (choix de contenu, audiences, etc.)

## Personnalisation

Avant d'utiliser un prompt, pensez à personnaliser :
- Vos choix de contenu marketing spécifiques
- Les audiences cibles
- Les objectifs de chaque plateforme
- Les outils et intégrations (APIs, CRMs, etc.)
- Le ton et le style de votre marque

## Principes Clés

Tous les prompts suivent ces principes :

1. **Valeur d'abord** : Chaque contenu doit apporter de la valeur authentique
2. **Adaptation plateforme** : Respecter la culture et les spécificités de chaque plateforme
3. **Engagement authentique** : Privilégier l'engagement authentique au marketing évident
4. **Optimisation continue** : Inclure des mécanismes d'analyse et d'itération
